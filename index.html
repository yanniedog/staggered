How this calculator works

Order Skew builds symmetric buy and sell ladders around a reference price, keeping a single source of truth so charts, tables, and exports always match.

  • Use Basic Setup to set capital, current price, ladder width, and skew.
  • Fees & Execution controls how trading fees are applied and how spacing between rungs is measured.
  • Advanced Options enable sell-only ladders and equal-quantity overrides for existing positions.

The calculator allocates capital across rungs, applies fees per your settings, then derives progressive average prices and net profit so the tables and charts represent the exact same plan.
